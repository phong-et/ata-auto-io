# ata-auto-io

## Service attendance

```js
node service.js
```

## Run as window service

```js
nssm install AttendanceService "C:\Program Files\nodejs\node.exe" "H:\VSCode\ata-auto-io\service.js"
```

## Server attendance controler

```js
npm start
```

## Client attendance controler

```js
node client
```
